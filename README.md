# RoutinyManager - مدير المهام الاحترافي

## 📱 نظرة عامة

**RoutinyManager** هو تطبيق احترافي لإدارة المهام مبني بـ Flutter مع تصميم عصري وميزات متقدمة. يتميز التطبيق بواجهة أنيقة بألوان الرمادي والفضي، ويدعم الوضع الليلي والنهاري، مع نظام إدارة مهام شامل.

## ✨ الميزات الرئيسية

### 🎨 التصميم
- **Theme متقدم**: دعم كامل للوضع الليلي والنهاري
- **تدرجات لونية احترافية**: رمادي وفضي مع لمسات من الألوان المميزة
- **واجهة مستخدم سلسة**: رسوم متحركة وانتقالات سلسة
- **تصميم Material 3**: أحدث معايير Google للتصميم

### 📋 إدارة المهام
- **إنشاء وتعديل المهام** بسهولة
- **4 مستويات أولوية**: منخفضة، متوسطة، عالية، عاجلة
- **8 فئات**: شخصي، عمل، تسوق، صحة، مالية، تعليم، اجتماعي، أخرى
- **تواريخ الاستحقاق** والتذكيرات
- **الوسوم (Tags)** لتنظيم أفضل
- **ملاحظات** لكل مهمة
- **حالات المهام**: نشطة، مكتملة، متأخرة، مجدولة لليوم

### 🔍 البحث والفلترة
- **بحث متقدم** في العناوين والأوصاف
- **فلاتر متعددة**: الكل، النشطة، المكتملة، اليوم، المتأخرة، عالية الأولوية
- **ترتيب ذكي**: حسب التاريخ، الأولوية، العنوان

### 📊 الإحصائيات والتحليلات
- **معدل الإنجاز** في الوقت الفعلي
- **إحصائيات شاملة**: المهام الكلية، المكتملة، المتبقية، المتأخرة
- **تحليل حسب الفئات** والأولويات
- **رسوم بيانية** تفاعلية

### 🔔 نظام الإشعارات
- **تذكيرات محلية** للمهام
- **جدولة دقيقة** للإشعارات
- **إشعارات يومية** قابلة للتخصيص

### 👤 إعدادات المستخدم
- **ملف شخصي** قابل للتخصيص
- **تبديل سهل** بين الوضع الليلي والنهاري
- **التحكم في الإشعارات**
- **خيارات الحذف**: حذف المهام المكتملة أو جميع البيانات

### 💾 التخزين
- **Hive Database**: قاعدة بيانات محلية سريعة وموثوقة
- **SharedPreferences**: للإعدادات والتفضيلات
- **حفظ تلقائي** لجميع التغييرات

## 🚀 التثبيت والتشغيل

### المتطلبات
- Flutter SDK 3.2.0 أو أحدث
- Dart SDK 3.0.0 أو أحدث
- Android Studio / VS Code
- محاكي Android/iOS أو جهاز فعلي

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone [repository-url]
cd RoutinyManager
```

2. **تثبيت الحزم**
```bash
flutter pub get
```

3. **تشغيل Build Runner** (لـ Hive)
```bash
flutter packages pub run build_runner build --delete-conflicting-outputs
```

4. **تشغيل التطبيق**
```bash
flutter run
```

## 🎯 كيفية الاستخدام

### إضافة مهمة جديدة
1. اضغط على زر "+" في الشاشة الرئيسية
2. أدخل عنوان المهمة (إلزامي)
3. أضف وصف، أولوية، فئة
4. حدد تاريخ الاستحقاق (اختياري)
5. فعّل التذكير إذا أردت
6. أضف وسوم وملاحظات
7. اضغط "حفظ"

### البحث والفلترة
- استخدم شريط البحث للبحث في المهام
- اضغط على الفلاتر للتصفية حسب الحالة
- يمكنك الجمع بين البحث والفلترة

### عرض الإحصائيات
- انتقل إلى تبويب "الإحصائيات"
- شاهد معدل الإنجاز والإحصائيات التفصيلية
- تابع تقدمك اليومي والأسبوعي

### تخصيص الإعدادات
- انتقل إلى تبويب "الحساب"
- غيّر اسمك، فعّل/عطّل الإشعارات
- بدّل بين الوضع الليلي والنهاري
- إدارة البيانات والحذف
